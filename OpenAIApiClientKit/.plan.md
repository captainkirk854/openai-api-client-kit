# Multi Model Consolidation

→ Takes a single user prompt
→ Fans out to N models
→ Produces a consolidated response

## Option 1: LLM As Judge
You feed all model outputs into a judge model (could be GPT‑4.1, o3-mini-high, or your preferred reasoning model) and ask:
- Which answer is most correct?
- Which is most complete?
- Which is most aligned with the user’s intent?

This gives you a deterministic, explainable selection.

## Option 2: Heuristic scoring (fast, cheap, deterministic)

Define a scoring function:
- Length
- Presence of hallucination markers
- Number of factual claims
- Confidence markers
- Domain‑specific keywords

This is useful when cost matters or when you want deterministic behavior without a judge model.

## Option 3: Response fusion

Instead of selecting one answer, you ask a judge model to synthesize the best elements of all answers into a single improved response.
This is extremely powerful for:
- Reasoning tasks
- Creative tasks
- Technical explanations
- Summaries
